# Server Template

This folder is a ready-to-use template for hosting your modpack backend.

## Quick Start

1. Copy your mod files into `files/` (BepInEx structure)
2. Edit `deploy.ps1` — set your URL and modpack name
3. Run `.\deploy.ps1`
4. Upload the entire `server-template/` folder to your web server
5. Give players the manifest URL

## Folder Structure

```
server-template/
├── manifest.json          ← generated by deploy.ps1
├── changelog.txt          ← edit this with update notes
├── deploy.ps1             ← one-click manifest generation
├── upload.ps1             ← optional: auto-upload via SCP/rsync
├── files/                 ← YOUR MOD FILES GO HERE
│   └── BepInEx/
│       ├── core/          ← BepInEx framework files
│       ├── plugins/       ← mod DLLs
│       └── config/        ← mod config files
├── launcher/
│   └── latest.json        ← optional: launcher self-update
├── nginx.conf             ← example nginx config
├── web.config             ← example IIS config
└── Caddyfile              ← example Caddy config
```

## What to give players

- **Manifest URL**: `https://YOUR-DOMAIN/valheim/manifest.json`
- **Server Host**: your Valheim server IP or hostname
- **Server Port**: `2456` (default)
- **Password**: tell them privately
